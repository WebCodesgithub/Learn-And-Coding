Cascading Style Sheet

CSS is used to set style of document written in markup language.Along with HTML and JavaScript,CSS is used to create interactive dynamic pages. The main difference between HTML and CSS is that HTML is used to structure the document and CSS is used to provide formatting to the structured document.

It enable the separation of document content from document presentation, including aspects such as the layout,colors and fonts. It also helps to improve content accessbility, more flexibility to change the presentation of the document. It helps to present the markup page in different styles.

There are three different types of style sheets in CSS :


1.Inline Style Sheet


When you want to apply the style on a particular HTML element, then this Style Sheet is being used. This overrides the style coded in Embedded or External Style Sheet.Let see the program.


<html>
<head>
<title>inline style sheet </title>
</head>
<body>
<h2 style="background-color:red;">this is heading </h2>
<p style="background-color:green;">this is a paragraph. </p>
</body>
</html>

Output:-

this is heading

this is paragraph



2. Internal Style Sheet


In this, in HTML document style sheet is being embedded using the (style)tag in the (head) section of the document. It is used when you want the same style to be repeated on one page of website, but those same styles are not required on entire website.Let see the program.


<html>
<head>
<title>internal style sheet </title>
<style type="text/css">
body
{
background-color:white;
}
h2
{
background-color:red;
}
p
{
background-color:pink;
color:blue;
font-weight:bold;
}
</style>
</head>
<body>
<h2>this is heading</h2>
<p>this is paragraph </p>
</body>
</html>

Output:-

this is heading

this is paragraph

In the community of web developers, JavaScript is highly popular as client side scripting language for web browsers. In any web application, JavaScript can be used to implement simple features like add animations into the page which help to attract to a specfic part of the page.
Provide responses within the web page to various actions performed by the users,load new images without needing to reload the entire page.

Features of JavaScript :

1.Browser supported


To access flash content, you need to install flash plugin in your browser. But to use JavaScript, you don't have to use any plugin at all. This is because all browsers have accepted JavaScript as a scripting language for them and provides integrated support for it. All you need to do is to handle to some of the tasks that are depent on DOM.


2.A Great Programming Tool For HTML



JavaScript is a powerful scripting language that helps HTML designers to effectively and interactively design websites and web pages in a very simple and efficient way.


3.Case Sensitive



JavaScript is a case-sensitive language. This means that the words can differ in meaning based on differing and lowercase letters. words with capital letter do not always have the same meaning when written with lowercase letters.
